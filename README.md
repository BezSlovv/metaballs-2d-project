# Metaballs (2D) — Marching Squares

Interactive 2D metaballs visualization built with **JavaScript**, **p5.js**, **HTML** and **CSS**.

## Demo

[Live Demo] https://bezslovv.github.io/metaballs-2d-project/

## Features

- animated 2D metaballs
- scalar field visualization
- Marching Squares isolines
- adjustable parameters
- click / tap to add metaballs
- responsive desktop and mobile layout

## Controls

- **Threshold**
- **Grid size**
- **Ball count**
- **Speed**
- **Source intensity**

Options:
- show grid
- show field points
- show scalar background
- show metaballs

Actions:
- start
- stop
- reset
- randomize
- add random
- remove last

## Notes

This project uses a scalar field generated by moving metaballs and extracts contours with the Marching Squares algorithm.  
Ambiguous cases are handled with center-based evaluation, and edge intersections are interpolated for smoother isolines.
